# 📘 Internship Project – AI Solutions Intern

## 📌 Overview

This repository contains the documentation and deliverables for my **8th Semester Internship** at **Raghuraj**.

The project focuses on building an **AI-powered chatbot system** using modern technologies such as **Generative AI, Retrieval-Augmented Generation (RAG), and full-stack web development**.

⚠️ **Note:** Due to company policies and NDA restrictions, the **source code is not included** in this repository.

---

## 🚀 Project Description

The internship project involves the design and development of an intelligent chatbot system capable of handling **technical queries related to sustainable energy solutions**.

The system leverages:

* Retrieval-Augmented Generation (RAG)
* Large Language Models (LLMs)
* Vector databases for semantic search

The chatbot provides:

* Real-time responses
* Context-aware answers
* Reduced dependency on manual support systems

---

## 🛠️ Technologies Used

* **Frontend:** React.js
* **Backend:** FastAPI
* **AI Framework:** LangChain
* **LLM:** OpenAI / Gemini
* **Database:** Vector Database (ChromaDB)
* **Version Control:** GitHub
